# Support

This repository is a starter for Spec Kit projects that want clean architecture,
test-first AI-assisted development, and language-neutral quality gates.

## Getting Help

Use GitHub issues for:

- Starter template bugs
- Documentation gaps
- Architecture guidance improvements
- Testing policy improvements
- Spec Kit workflow improvements

Use discussions, if enabled, for:

- Questions about applying the starter to a specific stack
- Examples of clean architecture structures
- Ideas for additional checklists or governance gates

## Before Asking

Check these files first:

- [README.md](README.md)
- [docs/spec-kit-workflow.md](docs/spec-kit-workflow.md)
- [docs/architecture.md](docs/architecture.md)
- [docs/testing-strategy.md](docs/testing-strategy.md)
- [docs/quality-gates.md](docs/quality-gates.md)

## Scope

This starter does not choose a language, framework, database, UI library, or AI
provider. Stack-specific setup belongs in the first feature plan after project
requirements are known.
