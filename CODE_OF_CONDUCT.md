# Code of Conduct

This project is intended to be useful to people with different backgrounds,
experience levels, tools, languages, and frameworks.

## Expected Behavior

- Be respectful and direct.
- Focus critique on code, architecture, tests, documentation, and decisions.
- Explain tradeoffs clearly.
- Welcome questions about the workflow and quality gates.
- Help contributors improve generated code instead of accepting weak output.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Publishing private information without permission.
- Dismissing review concerns without technical reasoning.
- Pressuring contributors to bypass tests, security checks, or architecture gates.

## Enforcement

Maintainers may edit, hide, reject, or remove contributions that violate this
code of conduct. Repeated or severe violations may lead to blocked participation.

Report concerns through the repository maintainers or the project's published
contact channel.
